Resources for developers

API

The Blue Button Connector provides a free, public, read-only API to access the directory of providers and receivers (apps).

API documentation & examples

Adding data to the Connector

Although the public API is currently read-only, you can use the web form to add a new provider or receiver. You'll find a link at the bottom of each provider profile page that you can use to suggest corrections to existing providers.

If your addition or correction is approved, it will appear on the website and in the API.

Connector website files

This Connector website has been built in the open. You can find the source files on Guthub.

Blue Button+

If you'd like to learn more about becoming a Blue Button provider or building an app that uses Blue Button data, check out the Blue Button+ website.
