Resources for developers

API

The Blue Button Connector provides a free, public, read-only API to access the directory of organizations and apps.

API documentation & examples

Adding data to the Connector

Although the public API is currently read-only, you can use the web form to add a new organization or app. You'll find a link at the bottom of each organization profile page that you can use to suggest corrections to existing organizations.

If your addition or correction is approved, it will appear on the website and in the API.

Data we're collecting for organizations

Note that some properties are not applicable to some categories.

Basics

  • name and description
  • contact and location
  • link to website
  • category

Additional features

  • Schedule or manage appointments
  • Send and receive secure messages
  • Manage health records for family members
  • Track and record self-reported health data
  • Search for a doctor in network
  • Dispute a claim
  • Shop for a plan
  • Refill your prescription
  • Set up automatic refills
  • Manage your family's prescription
  • Transfer your prescription
  • Request a new prescription

Download formats

  • text
  • PDF
  • C-CDA
  • C32
  • other

Data transmission

  • Project DIRECT (e.g. secure email)
  • Patient Trust Bundle (sharing data with patients/consumers)
  • Provider Trust Bundle (sharing data with other healthcare orgs)
  • Participation in other trust bundles

Viewable online

  • appointments
  • allergies
  • claims
  • clinical notes
  • diagnostics
  • family history
  • imaging
  • immunizations
  • lab results
  • medical history
  • medications
  • pathology
  • prescriptions
  • problems
  • visit history
  • vitals
  • BlueButton+ REST API (Pull) enabled
